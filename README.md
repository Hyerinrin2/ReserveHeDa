# ReserveHeDa

📅 스터디룸 예약 시스템

동시성 제어와 트랜잭션 처리를 중심으로 구현한 실시간 예약 시스템

📋 프로젝트 소개
여러 사용자가 동시에 같은 시간대를 예약할 때 발생하는 동시성 문제를 해결하고,
예약/취소 시 데이터 정합성을 보장하는 스터디룸 예약 시스템입니다.

🎯 개발 목표

JPA 낙관적 락을 활용한 동시성 제어
@Transactional을 통한 데이터 정합성 보장
RESTful API 설계 및 구현
실무 중심의 비즈니스 로직 처리

java21 </br>
SpringBoot 3.5.10 사용

✨ 주요 기능
1. 예약 관리

✅ 예약 가능 시간대 조회
✅ 예약 생성 (동시성 제어)
✅ 예약 조회 (본인 예약 목록)
✅ 예약 취소

2. 스터디룸 관리

스터디룸 목록 조회
스터디룸별 예약 현황 조회
